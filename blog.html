Monday, March 14, 2022

Hello, it's me, cooldude2144. sixteen days ago, i made an HTML game called FlappyHTML which is a flappy bird clone made using raw HTML, so recently, i decided to compare FlappyHTML to Flappybird.io and well, my game was weak, because Flappy Bird has graphics, flapping, a game over screen and a border. while my game has a square and a bunch of rectangles, no flapping and you can fly in the border. but as i said, FlappyHTML is nothing more than a prototype and i might use this prototype to work on something new and original using HTML, CSS or maybe JavaScript, and do something better than a red block floating above the pipes.
